Mobile Native Foundation

Improving engineering for large-scale mobile applications

What is the Mobile Native Foundation?

The Mobile Native Foundation is a collaboration of companies working toward improving processes and technologies related to rapidly and regularly releasing large-scale native Android and iOS applications built by large development teams. The Mobile Native Foundation provides a place to collaborate on open source projects and discuss wide ranging topics with industry leaders.
